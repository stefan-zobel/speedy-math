# speedy-math

Faster Java implementations of hypot, expm1, cos, sinh, cosh, tanh, asin, acos, atan and atan2

### Maven:

```xml
<dependency>
    <groupId>net.sourceforge.streamsupport</groupId>
    <artifactId>speedy-math</artifactId>
    <version>1.0.0</version>
</dependency>
```
